# Vue 3 + Tiptap ⚡

> 该仓库主要学习基于 Vue3 + Tiptap 封装一个富文本编辑器

<!-- [在线预览 github](https://xxx.github.io/xxxxx) -->

## 克隆代码到本地

```shell

# github
git clone https://github.com/htmlxudong/vue3-tiptap.git

# gitee
git clone https://gitee.com/xd_web/vue3-tiptap.git

```

## 安装依赖 📦

```
npm install

```

## 运行项目 🚀

```
npm run dev

```
